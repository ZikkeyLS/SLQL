# *SOQL*
SOCL - string oriented query language (based on SQL standarts)

## TIL – Table interaction language
* **CREATE** - create table;
* **EDIT** - edit table;
* **REMOVE** - remove table;
## ERL – Element redact Language
* **GET** – get element;
* **INSERT** – add element;
* **CHANGE** – change element;
* **DELETE** – remove element;
